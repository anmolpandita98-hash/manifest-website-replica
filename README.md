## Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/anmolpandita98-hash/manifest-website-replica.git
cd manifest-website-replica
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to:
```
http://localhost:3000
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Components

### ArticleCard
Displays articles in grid or list view with gradient backgrounds and animations.

### Navbar
Professional navigation bar with logo, menu links, search, and CTA button.

### Icons
Custom SVG icons including:
- TriangleIcon
- ThreeLobeIcon (Starfish)
- FourDotsIcon (Diamond)
- GridIcon
- ListIcon
- And more...

## Styling

This project uses Tailwind CSS for styling. Custom colors and theme extensions are configured in `tailwind.config.js`.
